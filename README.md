Hymate is a small mod that adds useful features just for Hypixel. It helps you play faster, be friendly, and have a smoother time on the server.

**What it does**
- Quick Join: Adds a Join Hypixel button to the title screen for quick access.
- Nametag: Displays your own name tag in-game.
- Click Action: Middle-click players to: add/remove as a friend, invite/remove from a party.
- AutoFriend: Automatically accept friend invites.
- AutoGG: Automatically sends “gg” at the end of every game.
- AutoTip: Automatically /tipall every 15 minutes.
- Queue Keybind: Set a favourite game to join instantly with one click.

![Mod settings screen.](https://cdn.modrinth.com/data/cached_images/94797be2f8b5b9e99da74f62ea9faa9a36aa1492_0.webp)

**How it works**
- Press Right Ctrl to open the settings menu.
- You can turn features on or off.
- You can change feature keybinds in the vanilla keybind menu.

**Requirements**
Fabric API (required) – Make sure you have this installed for the mod to work.
